Twounter returns the number of followers of a twitter user in simple text format.

== Description ==

Twounter simply returns the number of followers for a twitter username.

You have two options to fetch the number of followers for a username.

You can use the function `<?php echo twounter('username') ?>` in your template files or you can use the twounter 
shortcode `[twounter]username[/twounter]`in post/page editor to return the numbers of followers for the 
username.

== Installation ==

1. Upload `twounter` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can you use twounter shortcode on post/page editor? =

Yes, you can use the shortcode [twounter]username[/twounter] in post/page editor to get the number of 
followers for that username.
